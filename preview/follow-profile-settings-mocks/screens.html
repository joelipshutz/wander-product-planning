01 — MAP
Map / home
Map-first. No landing page. Reads as a trusted map within the first 5 seconds, never a social feed.
1

Search field

Place, vibe, area, contact, or username. Trailing avatar opens Profile. States: default, focused, loading, empty, error, active-filter summary.

w-searchradius.pillshadow-card
2

Filter chips · multi-select

you / friends / been / wanna go. Up to four pin states at once. Selection must not shift layout; readable by fill + label, not color alone.

w-chip.ondashed = wanna
3

Pins · owner + status

Owner = color (terracotta you / sky social), status = fill (solid been / dashed wanna). Category emoji inside. Small dot = distant/low-priority pin.

w-pin.youw-pin.social.wanna
4

Selected place sheet

Order: name + category + distance + been/wanna → social proof → top attributes → primary action → attribution. Collapsed / medium / expanded detents.

w-sheetradius.sheetshadow-sheet
5

Save / add to my map

Terracotta FAB on the sheet creates a UserPlace from a visible social place. Primary capture affordance also lives center-nav.

w-fabterracotta
6

Bottom nav · four tabs

map · add (+) · discover · profile. The plus is a tab, not a raised FAB — kept terracotta as the primary capture affordance. No world grouping labels. Your saved places & activity now live on profile; Settings opens from the gear there — not its own tab.

w-bottomnav4 tabs

04 — ADD · CONTEXTUAL QS
Contextual questions
Category-specific, not a generic review form. “This gets me.” Users can add custom tags per place.
1

Place context strip

Confirmed place echoed at the top with its been/wanna badge so the questions have an anchor. Never a card-in-card.

radius.md
2

Emoji scale

Tap-one rating. Selected gets a 2.5px terracotta ring. Type tag in mono labels the question type for the template system.

w-emoji-scaletype: emoji_scale
3

Multi-tag cloud

Category defaults (wifi solid, outlets, quiet…) plus user-added tags saved as future suggestions. Long tags truncate predictably.

w-tag.ontype: multi_tag
4

Note for future you

Free text. Quiet, optional, italic placeholder. Becomes the quote shown on the place sheet.

w-notetype: text
5

Save

Writes UserPlace + PlaceAttributes locally, enqueues sync. Optional answers never block this.

06 — PROFILE · YOU
Profile
The merged “you” surface: who you are + your place memory. Non-followers see only the header shell; followers see places per visibility.
1

Identity header card

Avatar, display name, “city · joined”, bio, and an edit action. This is the profile shell others see; place data below is gated by follow/visibility.

prof-cardtitle.lgsurface.bone
2

Stat tiles · been / wanna / friends

Tinted count tiles — terracotta / sun / sky. Tap-through to filtered place lists and the followers/following surfaces. Counts use tabular numerals.

.stat.been/.wanna/.friends
3

This month

Lightweight activity recap — check-in count, a plain-language summary, and a dotted streak strip. Not gamified; no badges or stre­ak pressure.

month-carddisplay.lg
4

Recent check-ins

Reverse-chron list of saved places: category thumb, name, area · time, and rating. Rows open the place sheet. Filter/sort lives at the section head later.

recent-rowradius.lg
5

Settings gear

The single entry to Settings (account, default visibility, blocked users, contacts, notifications). Not a bottom tab.
