00 — BRAND

Useful before social

Map first, not list first. Trusted people, not strangers. Capture must feel faster than organizing. Privacy must be visible before social value expands. Copy is informal but never slangy.

05 — PIN SYSTEM

Pin color system

Owner sets color, status sets fill. Up to four pin states show at once. VoiceOver labels include owner, status, category, and place name — e.g. “Social place, been, coffee, Maru Coffee.”
